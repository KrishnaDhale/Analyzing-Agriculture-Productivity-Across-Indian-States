# Analyzing-Agriculture-Productivity-Across-Indian-States
 
A Power BI project analyzing agricultural productivity across Indian states. The dashboard visualizes crop-wise and region-wise performance, highlights top and bottom regions, and provides insights for better decision-making.

🛠️ Tools Used

Power BI – Dashboard & KPIs

Excel/CSV – Dataset

PowerPoint – Project presentation

🚀 Key Insights

Identified low-performing regions for policy support

Highlighted high-yield crops for resource allocation

Showcased how visualization improves decision-making
